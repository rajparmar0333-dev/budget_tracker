# Budget Tracker (C Project)

A simple command-line based budget tracker written in C.  
It allows you to record daily expenses, categorize them, calculate totals, and generate basic summaries.  
The project is designed as a practice exercise in procedural C programming, input handling, and basic data processing.

---

##  Features

- Add daily expenses (food, travel, personal, etc.)
- Set monthly or weekly limits
- Calculate total spending
- Compare actual spending with limits
- Generate a summary report at the end
- Menu-driven CLI interface


---

##  How to Compile

Make sure you have GCC installed.

### **Linux / macOS**
```bash
gcc project.c -o budget_tracker
./budget_tracker
